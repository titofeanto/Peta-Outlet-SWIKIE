🚀 Atau Publikasikan Langsung ke GitHub

Lewati langkah download-upload manual — file langsung terkirim ke repository Anda. Perlu Personal Access Token dari GitHub (dibuat sekali, hanya untuk repo ini). Token tidak pernah disimpan di file atau di perangkat ini — hanya dipakai sesaat di browser dan hilang saat halaman ditutup/di-refresh.
Cara membuat token (sekali saja, ± 2 menit)
  1. Buka github.com/settings/tokens?type=beta, login ke akun GitHub Anda.
  2. Klik Generate new token.
  3. Isi nama bebas, misal Swikie Admin Publish.
  4. Expiration: pilih durasi singkat, misal 90 hari (lebih aman, nanti tinggal buat baru lagi).
  5. Repository access → pilih Only select repositories → centang repo peta Anda saja.
  6. Permissions → Repository permissions → cari Contents → ubah ke Read and write.
  7. Klik Generate token → salin tokennya (hanya muncul satu kali!) → tempel ke kolom Token di bawah.
Jangan bagikan token ini ke siapa pun — token ini seperti password khusus untuk repo Anda.
