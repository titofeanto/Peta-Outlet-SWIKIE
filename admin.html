Publikasikan Langsung ke GitHub

Cara membuat token (sekali saja, ± 2 menit)
  1. Buka github.com/settings/tokens?type=beta, login.
  2. Klik Generate new token.
  3. Nama: Swikie Admin. Expiration: 90 hari.
  4. Repository access → Only select repositories → centang repo peta.
  5. Permissions → Contents → Read and write.
  6. Generate → salin tokennya.
Jangan bagikan token ini ke siapa pun.
